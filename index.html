Monitoreo de servidores

Descripción

Nuestro proyecto busca mejorar la seguridad y el rendimiento de los servidores mediante un sistema de monitoreo de temperatura con sensores conectados a Micro:bit. El sistema alerta si la refrigeración falla o si la temperatura supera los límites establecidos.

Objetivo del Proyecto

Monitorear en tiempo real la temperatura dentro de los cuartos de servidores.
Detectar variaciones críticas que puedan afectar el rendimiento de los equipos.
Confirmar que el sistema de refrigeración funciona de manera adecuada.
Minimizar riesgos de sobrecalentamiento y fallas en los servidores.
Implementar una solución de bajo costo y fácil instalación..

Lista de Materiales

Arduino ESP32

Microcontrolador con conexión WiFi y Bluetooth, ideal para proyectos IoT y procesamiento de datos en tiempo real.

Fuente de energía

Dispositivo encargado de suministrar la energía necesaria para el funcionamiento del circuito y sus componentes.

Pantalla LED

Permite mostrar información como la temperatura, humedad o mensajes de estado en tiempo real.

Protoboard

Tablero de pruebas que facilita el montaje de circuitos sin necesidad de soldadura.

Sensor DHT11

Sensor digital que mide temperatura y humedad, enviando los datos al microcontrolador para su procesamiento.
