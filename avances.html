Avances

Avances por Semana

Semana 1 (15/08 – 21/08)

Durante esta primera semana se realizó la puesta en marcha del proyecto, comenzando por la definición de los objetivos principales y el alcance esperado. Se procedió a la selección del hardware base, optando por el microcontrolador ESP32 como unidad central y el sensor de temperatura DS18B20 para las primeras pruebas de captura de datos. Finalmente, se llevó a cabo una prueba inicial de lectura de temperatura, confirmando la correcta comunicación entre el sensor y el microcontrolador.

Semana 2 (22/08 – 28/08)

En esta etapa se abordó la construcción de la interfaz web. Se exploraron distintas alternativas de desarrollo, evaluando el uso de Google Sites y WordPress. Sin embargo, tras analizar sus limitaciones, se decidió implementar la página de forma manual con HTML y CSS, lo que brinda mayor flexibilidad para la integración futura de datos.
Definición de la estructura inicial para mostrar los datos del sensor.

Semana 3 (29/08 – 04/09)

Compra de materiales: ESP32 USB-C, DHT11, pantalla OLED 0.96", protoboard 400 puntos.

Recepción de jumpers proporcionados por el profesor.

Finalización de la página web lista para recibir e integrar datos del sensor.

Instalación del Arduino IDE para programar el ESP32.

Preparación del prototipo para integrar sensor, pantalla OLED y envío de datos.

Problemas iniciales al reconocer el ESP32 en Arduino IDE, se probaron diferentes configuraciones.

Entrega del primer avance al profesor: prototipo, página web y plan de trabajo.

Lista de Materiales

Arduino ESP32

Microcontrolador con conexión WiFi y Bluetooth, ideal para proyectos IoT y procesamiento de datos en tiempo real.

Fuente de energía

Dispositivo encargado de suministrar la energía necesaria para el funcionamiento del circuito y sus componentes.

Pantalla LED

Permite mostrar información como la temperatura, humedad o mensajes de estado en tiempo real.

Protoboard

Tablero de pruebas que facilita el montaje de circuitos sin necesidad de soldadura.

Sensor DHT11

Sensor digital que mide temperatura y humedad, enviando los datos al microcontrolador para su procesamiento.
